bi
-------

bi is collinearity and Ks speculate whole genome duplication.
  
.. rubric:: Parameters

Use command to enter the folder ``wgdi -bk ? > blockks.conf`` Take out the parameter file.::

   [b1ockinfo]
   blast = blast file
   gff1 = gffl file
   gff2 = gff2 file
   len31 = lens1 file
   1en32 = lens2 file
   colinearity = colinearity file
   score = 100
   evalue = 1e-5
   repnum = 30
   poaition = order
   ks = ks file
   ks_col = ka_ NG86
   savefile = block information (*.csv)
